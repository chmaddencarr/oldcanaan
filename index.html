Native River-Washed Stone, 3/8": The Only Gravel for a New England Driveway

The gravel is native river-washed stone, 3/8" grade, in a mix of warm grey, buff, and pale rose. It compacts loosely underfoot. It drains perfectly. It has been used on Connecticut driveways for as long as there have been Connecticut driveways.
